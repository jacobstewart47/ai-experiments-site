By Jake Stewart

Figma Cursor Chat GPT Github Vercel

What is it?

A small microsite to capture everything we built during a design team hack day. My main aim was to properly start using tools that would allow me to go from Figma to a fully deployed site. I started designing in Figma, and used their MCP server to get building in cursor. I used ChatGPT to help me debug any issues and used the terminal to start pushing to GitHub. After linking my Github repo with a new Vercel project, any changes I pushed would be live instantly.

What was the biggest challenge?

Using the terminal felt like a learning curve. I had to spend some time debugging some issues and I wasn’t familiar with creating and SSH so I could push to Github. I screenshotted any errors and got ChatGPT to walk me through how to solve them.

What was the biggest learning?

Using the Cursor and Figma’s MCP serve is really effective. I probably could have stayed in Figma a bit longer and made sure I had all the state and auto layout mapped out. Jumping into Cursor too quickly sometimes means you end up switching between tools when building something new. Keeping things organised in Figma helps!
