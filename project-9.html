Interactive prototype of the cost objects table

By Davi Magalhães

Figma Make

I've used Figma Make to create an interactive prototype of the cost objects table because it will feature a few things we don't have today, such as nested rows (because one cost object can be inside another cost object, creating a hierarchy among them). The one in the video is the tidiest version, because I've tried a few different things visually but it was impossible to get aligned lines (as you can see in the screenshot). I've also explored a few different ways to allow users to do bulk actions, as this will likely be a big thing too. All in all, it's great to see try out some of these complex interactions in a working prototype.
